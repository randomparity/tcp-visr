TCP Flow Visualizer TUI
=======================
A Rust TUI utility that helps visualize TCP flow data for a running system
or a from TCP dump captures.

Provides similar visual information to the TCP simulator:
https://github.com/Overshoot-ai/tcp-visualizer

Planned features:
- Variable time progression (shows real time data, replays at 0.1X to 10X)
- Others TBD
